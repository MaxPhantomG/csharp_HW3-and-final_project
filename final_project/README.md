**TaskHub** - это консольное приложение на C#, предназначенное для управления задачами. С помощью этого приложения вы можете добавлять, изменять, удалять и отслеживать задачи, а также получать статистику о выполненных и ожидающих задачах.

## МЕНЮ

* Добавить задачу (Add Task) - позволяет пользователю создать новую задачу, указав заголовок, описание и приоритет.

* Показать все задачи (Show All Tasks) - отображает список всех задач, включая их статус и приоритет.

* Показать выполненные задачи (Show Completed Tasks) - аыводит на экран только задачи, которые были выполнены.

* Показать невыполненные задачи (Show Pending Tasks) - отображает список задач, которые еще не завершены.

* Показать задачи с высоким приоритетом (Show High Priority Tasks) - фильтрует задачи по высокому приоритету и отображает их.

* Jбновить задачу (Update Task) - позволяет внести изменения в существующую задачу. Пользователь может обновить заголовок, описание или статус задачи.

* Удалить задачу (Remove Task) - позволяет пользователю удалить указанную задачу из списка задач.

* Поиск по заголовку (Search By Title) - позволяет искать задачи по заголовку, выводя на экран все соответствующие результаты.

* Поиск по статусу (Search By Status) - позволяет находить задачи по их статусу - выполненные или невыполненные.

* Поиск по приоритету (Search By Priority) - позволяет отфильтровать задачи по уровню приоритета: низкий, средний или высокий.

* Статистика (Statistics) - предоставляет обобщенную информацию о задачах: количество выполненных, невыполненных задач, а также количество задач по каждому уровню приоритета.

* Cохранить задачи (Save Tasks) - сохраняет текущий список задач в файл для последующего восстановления.

* Выход (Exit)
